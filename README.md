# Autonomous Mesh Intelligence System

A fault-tolerant distributed mesh intelligence system
for real-time node monitoring and resilient communication.

Traditional sequential communication systems fail
when a single intermediate node goes offline.

This project demonstrates how mesh-based
distributed systems improve fault tolerance
through real-time node monitoring.

## Real-Life Use Case

Disaster recovery communication networks
during earthquakes, floods, and emergencies.

## Features

- Real-time node telemetry
- MQTT communication
- Node failure simulation
- Offline detection
- Recovery state tracking
- Cloud deployment
- Live monitoring dashboard
